# ausbernard.github.io 🌱

Personal portfolio site. Hand-coded, no frameworks, no build step.

## 🗂 What's in here

- **`index.html`** — home
- **`about.html`** — who I am, stack, CV
- **`uses.html`** — tools/gear
- **`projects/`** — repos + open source contributions
- **`writing/`** — blog posts
- **`work/`** — engineering impact case studies
- **`assets/`** — CV + everything else

## 🛠 Stack

Plain HTML + CSS. That's it. Deployed via GitHub Pages.

## 🚀 Running locally

```bash
open index.html
```

or

```bash
spin up a Live Server
```
